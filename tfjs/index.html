Stock Prediction

Hello! Welcome to my stock predicting bot that was trained using Tensorflow and deployed using Tensorflow JS!

Feel free to poke around the source code by inspecting element.


How does the model work?

30 days of stock price → normalization and tanh activation → dense neural network → buy, sell, and hold "confidences"


How was the neural network trained?

This project implemented a deep q-networks, which was trained by replaying historical stock data to find patterns
